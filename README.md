# **DebtWizard**

DebtWizard is a web-based debt management platform that helps users organize debts, track payments, and provide debt repayment strategies.

## Features
- Debt Management
- Payment Tracking
- Interest Calculation
- Debt Summary
- DTI Analysis
- Apply multivariate statistical analysis to understand user behavior and find key factors influencing debt repayment performance(Planned)
- AI Recommendation (Planned)
- Gamification (Planned)

## Tech stack
- **Backend:** Spring Boot, Spring Data JPA
- **Database:** PostgreSQL
- **Authentication:** Cookie-based Session
- **Statistical Analysis:** R
- **Build Tool:** Maven
- **Version Control:** GitHub
